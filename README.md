# Meowfetch 🐾

**Meowfetch** is a cat-themed command-line tool that displays cute ASCII art of cats along with your system information, inspired by Neofetch.

## Features

- 🐱 Adorable cat ASCII art
- 📊 System info display
- 🎨 Customizable appearance
